Inspiration, Sources and Credits

Some of the functionality and assets on this website has been produced under insppiration from multiple sources.
Below is a list of assets and sources we used for this project to come to life.
We thank the original creators for their work and appreciate the hard work they put into these assets.

  • SpaceY Logo - Original SVG file / code from SpaceX Modified slightly to make the X look more like a Y.
  • Favicon - Modified version of SpaceX' favicon, downloaded from SpaceX Static Assets (Linked in their stylesheet).
  • D-DIN Fonts - Font files sourced from SpaceX' Stylesheet. Files hosted on SpaceX' servers.
  • Android Assassin Font - Font used on index.html header. Downloaded from Fonts2U.
  • Buttons (as a lot of other things on this website) inspired by SpaceX, code help from Giana on CodePen. Modified slightly for our use.
  • Form Country List - To save time, this list was copied from danrovito on GitHub.
  • Form Background Image downloaded from Teslarati.com.
  • Rocket Launch Sound (home.html) - Audio originally from Artemis Launch, Downloaded from YouTube.
  • Rocket Landing Sound (home.html) - Audio originally from a SpaceX Falcon Landing, Downloaded from YouTube.
  • ChatGPT was also used for certain parts to troubleshoot or better understand concepts for animations and placements using CSS.
  • GitHub Copilot also helped fill in some content on the Liability page (Autofilled some list suggestions)

It should come to no surprise that this website is heavilly inspired by SpaceX' website design.
However, all code was written by us except for the SpaceX/Y SVG and the country list on the reservation form.
We thank SpaceX for having such a fun design on their website. We had a great time trying to recreate it to the level we did.
